# تطبيق الرعوي

تطبيق **الرعوي** هو حل Flutter متكامل لإدارة حركة القات بين حسابات الرعوي، المقوت، والوكيل. يعتمد التطبيق على قاعدة بيانات SQLite محلية مع إدارة حالة بواسطة Provider، ويتضمن شاشة للحسابات، شاشة البيع اليومي، وشاشة قيد اليومية بالإضافة إلى نظام تفعيل لمدة 360 يومًا بعد إدخال 20 عملية.

## المزايا الرئيسية
- إدارة ثلاثة أنواع من الحسابات (رعوي، مقوت، وكيل) مع اسم مستخدم وكلمة مرور مدمجين في بطاقة الحساب.
- تصفية الحسابات بحسب الاسم أو رقم الهاتف مباشرة من شاشة الحسابات.
- تسجيل عمليات البيع اليومية مع حساب العمولة (5٪) وإظهار صافي المستحق للرعوي.
- سجل يومي (قيود) يعرض تفاصيل القيد لكل عملية بما في ذلك إجمالي البيع، العمولة، والصافي.
- عرض جميع عمليات كل حساب عند الضغط على زر "البيع اليومي" مع إمكانية إضافة عملية جديدة من نفس الشاشة.
- نظام تفعيل تلقائي: بعد 20 عملية يتوقف الإدخال إلى أن يتم تفعيل التطبيق بواسطة رمز ينتجه المدير، والتفعيل صالح لمدة 360 يومًا.

## البدء
1. ثبّت Flutter 3.10 أو أحدث.
2. ثبّت الحزم:
   ```bash
   flutter pub get
   ```
3. شغّل التطبيق:
   ```bash
   flutter run
   ```

> **ملاحظة:** التطبيق يعتمد على حزم `provider`, `sqflite`, و `intl` بالإضافة إلى `flutter_localizations` لتقديم واجهة عربية بالكامل.
